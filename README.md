## Components

 * 3 - Unkown Continious Rotation Servo Motor;
 * 1 - Bluetooth RS232 HC-05;
 * 1 - 7805CT;
 * 3 - Ultrasonic Sensor;
 * 1 - Arduino UNO;
 * 3 - Onmidirectional Wheels;
 * 1 - Protoboard;
 * Chassi & Jumpers.
